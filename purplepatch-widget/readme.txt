== Description ==

PurplePatch Widget is a WordPress plugin that provides functionality for image optimization and other features from Purplepatchservices.

== Features ==

* **Image Optimization**: Automatically resizes and optimizes images.
* **Image Optimization Reports**: View detailed reports on image optimization, including total images, optimized images, and disk space saved.
* **Additional Settings**: Configure image optimization settings and schedule tasks.

== Installation ==
1. Upload the `purplepatch-widget` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the settings page and configure the options as needed.
